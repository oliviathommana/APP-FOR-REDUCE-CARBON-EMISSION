# ⚡ EV EcoDrive - Mobile Application

An innovative mobile application designed to encourage the adoption of Electric Vehicles (EVs) by providing smart features that improve the user experience while contributing to a reduction in carbon emissions.

## 🌍 Project Objective

The increasing use of fossil-fuel-powered vehicles contributes significantly to carbon emissions. This application aims to support sustainable transportation by offering EV users an all-in-one platform for charging, trip planning, and environmental impact tracking.

---

## ✨ Features

- 🔋 Locate nearby EV charging stations
- 🗺️ Smart route planning
- 📊 Battery status monitoring
- 🌱 Carbon emission savings tracker
- 📍 Live GPS navigation
- 🔔 Charging reminders and notifications
- 👤 User profile management
- 📈 Trip history and analytics

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/ev-ecodrive.git
```

2. Open the project

```bash
cd ev-ecodrive
```

3. Install dependencies

Flutter

```bash
flutter pub get
```

React Native

```bash
npm install
```

4. Run the application

Flutter

```bash
flutter run
```

React Native

```bash
npm start
```
